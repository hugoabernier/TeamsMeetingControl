# TeamsMeetingControl CLI

Diagnostic console utility for Teams Keyboard Controls. The CLI reuses `TeamsShortcuts.Core`, which is also used by the Stream Deck plugin.

```powershell
dotnet run --project .\src\TeamsShortcuts.Console\TeamsShortcuts.Console.csproj -- diagnose --verbose
dotnet run --project .\src\TeamsShortcuts.Console\TeamsShortcuts.Console.csproj -- hand
dotnet run --project .\src\TeamsShortcuts.Console\TeamsShortcuts.Console.csproj -- react laugh
```
